## Problem Summary
Banks manually review thousands of loan applications, taking 3-7 days per decision. This AI solution uses a Feed-Forward Neural Network to predict loan default risk in real-time, reducing processing time to under 1 hour while maintaining human oversight for complex cases.

| Task 2 | Business problem defined: Manual loan approval is slow and inconsistent |
| Task 3 | AI task identified: Binary Classification (Default vs No Default) |
| Task 4 | Data requirement plan: 14 input features, historical loan records |
| Task 5 | Model recommended: Feed-Forward Neural Network |
| Task 6 | Evaluation plan: AUC-ROC, F1 Score, business metrics defined |

## Key Highlights

- **AI Task:** Binary Classification
- **Model:** Feed-Forward Neural Network (Input→64→32→Output)
- **Expected Impact:** 20-30% reduction in defaults, 90% faster processing
- **Responsible AI:** SHAP explainability, bias audits, RBI compliance
